# Laundry Wallah / Laundry Mart

## Description
This is a simple HTML webpage for Laundry Wallah / Laundry Mart, a professional laundry service provider. The page showcases the company's services, pricing, and allows users to book laundry services online.

## Features
- **Header Section**: Displays a welcome message for the laundry service.
- **About Us**: Provides an overview of the company, emphasizing convenience, quality, and care for customers' clothes.
- **Image Section**: Includes a relevant laundry image to enhance visual appeal.
- **Services**: Lists the various laundry services offered, such as washing, ironing, dry cleaning, stain removal, express services, and special garment care.
- **Price List**: A table detailing the prices for each service (in Indian Rupees).
- **Book Now**: A simple form for users to enter their name and email to book services.

## Technologies Used
- HTML (with basic form elements and table structure)

## How to View
1. Download or clone the project files.
2. Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge).
3. Navigate through the sections using the navigation menu or by scrolling.

## Navigation
The page includes a navigation bar with links to:
- About Us
- Our Services
- Price List
- Book Now
- Contact Us (links to footer)

## Contact
For inquiries, refer to the contact information in the footer section of the webpage.
